# Speech-vs-Plan

Система автоматического анализа соответствия устной речи преподавателя заявленному плану лекции. Проект разрабатывается в рамках командной учебной практики.

## 🎯 Описание проблемы и бизнес-ценность проекта
Часто значимые проектные решения, лекционный материал или договоренности озвучиваются устно во время занятий или созвонов. При этом возникает разрыв между тем, что планировалось обсудить (agenda / план занятия), и тем, что реально было сказано. Преподавателям и менторам сложно вручную контролировать полноту выданного материала. 

Проект **Speech-vs-Plan** решает эту проблему: система будет автоматически сопоставлять аудиозаписи выступлений с их изначальными планами, выявляя пропущенные темы, незапланированные дополнения (экспромты) и фиксируя обещания/задачи (Next Actions).
